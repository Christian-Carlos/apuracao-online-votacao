# Apuração Online de Votação

## Descrição
Sistema de apuração de votos em tempo real desenvolvido com Google Apps Script e Google Sheets. Responsável por processar os dados da votação, gerar rankings automáticos e apresentar os resultados de forma clara e organizada.

---

## Funcionalidades
- Apuração automática de votos
- Ranking em tempo real
- Classificação dos mais votados
- Identificação de suplentes
- Integração direta com base de dados da votação
- Atualização dinâmica dos resultados

---

## Tecnologias
- Google Apps Script
- JavaScript
- HTML
- CSS
- Google Sheets

---

## Diferenciais
- Atualização em tempo real
- Estrutura otimizada para grandes volumes de votos
- Transparência na apuração
- Organização automática dos resultados
- Pronto para uso institucional

---

## Observação
Este sistema é complementar ao repositório **sistema-votacao-gas**, sendo responsável exclusivamente pela apuração e visualização dos resultados.
